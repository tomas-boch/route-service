# route-service

## build
    cd route-service
    mvn clean install

## run
Required Java version is 25.

    cd route-service/target
    java -jar ./route-service-0.0.1-SNAPSHOT.jar

Or run *RouteApplication* class in your IDE (Idea). 

## usage
1. in your browser go to e.g. http://localhost:8080/routing/LSO/SWE to find a route from Lesotho to Sweden

2. check the log file logs/route-service.log
